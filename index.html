How do I use them?

It’s easy! Once you’ve added the Node package you’ll have access to hundreds of icons that you can easily customize.

import { Bitcoin } from 'grommet-icons';
<Bitcoin color='plain' size='xlarge' /> 
<Bitcoin size='large' /> 
<Bitcoin color='brand' /> 
<Bitcoin />
